=============
Prerequisiti
=============

- Lightroom 6+ o CC classic
- Windows 7+ o MacOS 10.12+
- Un controller MIDI USB (ad esempio: Korg nanoKONTROL, ecc.)

=====
Impostare
=====

1. Scarica il programma di installazione per il tuo sistema operativo da https://github.com/rsjaffe/MIDI2LR/releases, dove troverai sempre l'ultima versione. Quando selezioni una cartella per l'installazione, assicurati di non aver bisogno dei privilegi di amministratore per scrivere in quella cartella; il plug-in potrebbe bloccarsi se è in esecuzione in una cartella con restrizioni.

2. In Lightroom, aggiungi il plugin MIDI2LR andando su File → Gestione plugin → Aggiungi e navigando e selezionando la cartella MIDI2LR.lrplugin.

3. L'applicazione MIDI2LR dovrebbe avviarsi automaticamente. Se è chiuso, può essere avviato andando su File → Plug-in Extra → Avvia MIDI2LR

4. Lo spostamento di qualsiasi controller popolerà la tabella di mappatura e consentirà di impostare le funzioni. Puoi anche salvare / caricare i mapping.

5. Le informazioni su tutte le impostazioni e le opzioni sono nel wiki: https://github.com/rsjaffe/MIDI2LR/wiki.

================================
Aggiornamento da una versione precedente
================================

1. Chiudi Lightroom.

2. Installa nella stessa directory utilizzata nella versione precedente.

3. Riavvia Lightroom.
