=============
先決條件
=============

-Lightroom Classic 11+
-Windows 10以上版本或MacOS 12.0以上版本
-USB MIDI控制器（例如：Korg nanoKONTROL等）

=====
設定
=====

1.從https://github.com/rsjaffe/MIDI2LR/releases下載適用於您的操作系統的安裝程序，您將始終在其中找到最新版本。選擇要安裝的文件夾時，請確保您不需要管理員權限才能寫入該文件夾-如果插件在受限文件夾中運行，則插件可能會崩潰。

2.在Lightroom中，通過轉到文件→插件管理器→添加並導航到並選擇MIDI2LR.lrplugin文件夾來添加MIDI2LR插件。

3. MIDI2LR應用程序應自動啟動。如果關閉，可以通過轉到文件→插件附加→啟動MIDI2LR來啟動它。

4.移動任何控制器將填充映射表，並允許您設置功能。您還可以保存/加載映射。

5.有關所有設置和選項的信息，請參見Wiki：https：//github.com/rsjaffe/MIDI2LR/wiki。

===============================
從以前的版本更新
===============================

1.結束 Lightroom。

2.安裝到與先前版本相同的目錄。

3.重新啟動Lightroom。
