=============
ข้อกำหนดเบื้องต้น
=============

- Lightroom 6+ หรือ CC classic
- Windows 7+ หรือ MacOS 10.12+
- คอนโทรลเลอร์ USB MIDI (เช่น Korg nanoKONTROL และอื่น ๆ )

=====
ติดตั้ง
=====

1. ดาวน์โหลดตัวติดตั้งสำหรับระบบปฏิบัติการของคุณจาก https://github.com/rsjaffe/MIDI2LR/releases ซึ่งคุณจะพบเวอร์ชันล่าสุดเสมอ เมื่อเลือกโฟลเดอร์สำหรับการติดตั้งตรวจสอบให้แน่ใจว่าคุณไม่จำเป็นต้องมีสิทธิ์ของผู้ดูแลระบบในการเขียนไปยังโฟลเดอร์นั้นปลั๊กอินอาจมีปัญหาหากทำงานในโฟลเดอร์ที่ถูก จำกัด

2. ใน Lightroom เพิ่มปลั๊กอิน MIDI2LR โดยไปที่ File →ตัวจัดการปลั๊กอิน→เพิ่มและไปที่และเลือกโฟลเดอร์ MIDI2LR.lrplugin

3. แอปพลิเคชั่น MIDI2LR ควรเปิดขึ้นโดยอัตโนมัติ ถ้ามันถูกปิดก็สามารถเปิดได้โดยไปที่ File → Plugin Extras → Start MIDI2LR

4. การย้ายตัวควบคุมใด ๆ จะเติมตารางการแมปและอนุญาตให้คุณตั้งค่าฟังก์ชั่น นอกจากนี้คุณยังสามารถบันทึก / โหลดการแมป

5. ข้อมูลเกี่ยวกับการตั้งค่าและตัวเลือกทั้งหมดอยู่ในวิกิ: https://github.com/rsjaffe/MIDI2LR/wiki

================================
การอัปเดตจากรุ่นก่อนหน้า
================================

1. ออกจาก Lightroom

2. ติดตั้งไปยังไดเรกทอรีเดียวกันกับที่ใช้ในเวอร์ชันก่อนหน้า

3. รีสตาร์ท Lightroom
