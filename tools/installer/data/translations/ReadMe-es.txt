=============
Prerrequisitos
=============

- Lightroom Classic 11+
- Windows 10 o MacOS 10.15+
- Un controlador MIDI USB (por ejemplo: Korg nanoKONTROL, etc.)

=====
Preparar
=====

1. Descargue el instalador para su sistema operativo desde https://github.com/rsjaffe/MIDI2LR/releases, donde siempre encontrará la última versión. Al seleccionar una carpeta para la instalación, asegúrese de que no necesita privilegios de administrador para escribir en esa carpeta; el complemento puede bloquearse si se ejecuta en una carpeta restringida.

2. En Lightroom, agregue el complemento MIDI2LR yendo a Archivo → Administrador de complementos → Agregar y navegando y seleccionando la carpeta MIDI2LR.lrplugin.

3. La aplicación MIDI2LR debería iniciarse automáticamente. Si está cerrado, puede iniciarse yendo a Archivo → Extras de complementos → Iniciar MIDI2LR

4. Mover los controladores llenará la tabla de mapeo y le permitirá establecer funciones. También puede guardar / cargar asignaciones.

5. La información sobre todas las configuraciones y opciones se encuentra en la wiki: https://github.com/rsjaffe/MIDI2LR/wiki.

================================
Actualización desde una versión anterior
================================

1. Salir de Lightroom.

2. Instale en el mismo directorio que se utilizó en la versión anterior.

3. Reinicie Lightroom.
