=============
전제 조건
=============

-Lightroom 11 이상 또는 CC 클래식
-Windows 10 이상 또는 MacOS 10.15 이상
-USB MIDI 컨트롤러 (예 : Korg nanoKONTROL 등)

=====
설정
=====

1. 항상 최신 버전을 찾을 수있는 https://github.com/rsjaffe/MIDI2LR/releases에서 운영 체제 설치 프로그램을 다운로드하십시오. 설치할 폴더를 선택할 때 해당 폴더에 쓰기 위해 관리자 권한이 필요하지 않은지 확인하십시오. 제한된 폴더에서 플러그인을 실행하면 플러그인이 중단 될 수 있습니다.

2. Lightroom에서 File → Plugin-Manager → Add로 이동하여 MIDI2LR.lrplugin 폴더로 이동하여 선택하여 MIDI2LR 플러그인을 추가합니다.

3. MIDI2LR 응용 프로그램이 자동으로 시작됩니다. 닫혀 있으면 File → Plug-in Extras → Start MIDI2LR로 이동하여 시작할 수 있습니다.

4. 컨트롤러를 이동하면 매핑 테이블이 채워지고 기능을 설정할 수 있습니다. 매핑을 저장 /로드 할 수도 있습니다.

5. 모든 설정 및 옵션에 대한 정보는 wiki (https://github.com/rsjaffe/MIDI2LR/wiki)에 있습니다.

=================================
이전 버전에서 업데이트
=================================

1. Lightroom 종료.

2. 이전 버전에서 사용한 것과 동일한 디렉토리에 설치하십시오.

3. Lightroom을 다시 시작하십시오.
