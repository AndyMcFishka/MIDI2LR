=============
Voraussetzungen
=============

- Lightroom Classic 11+
- Windows 10+ oder MacOS 10.15+
- Ein USB-MIDI-Controller (z. B. Korg nanoKONTROL usw.)

=====
Konfiguration
=====

1. Laden Sie das Installationsprogramm für Ihr Betriebssystem von https://github.com/rsjaffe/MIDI2LR/releases herunter. Dort finden Sie immer die neueste Version. Stellen Sie bei der Auswahl eines Ordners für die Installation sicher, dass Sie keine Administratorrechte zum Schreiben in diesen Ordner benötigen. Das Plugin kann abstürzen, wenn es in einem eingeschränkten Ordner ausgeführt wird.

2. Fügen Sie in Lightroom das MIDI2LR-Plugin hinzu, indem Sie zu Datei → Plugin-Manager → Hinzufügen gehen und zum Ordner MIDI2LR.lrplugin navigieren und diesen auswählen.

3. Die MIDI2LR-Anwendung sollte automatisch gestartet werden. Wenn es geschlossen ist, kann es über Datei → Plug-in-Extras → MIDI2LR starten gestartet werden

4. Durch Verschieben von Controllern wird die Zuordnungstabelle gefüllt und Sie können Funktionen festlegen. Sie können Zuordnungen auch speichern / laden.

5. Informationen zu allen Einstellungen und Optionen finden Sie im Wiki: https://github.com/rsjaffe/MIDI2LR/wiki.

===============================
Aktualisieren von einer früheren Version
===============================

1. Lightroom beenden.

2. Installieren Sie in demselben Verzeichnis wie in der vorherigen Version.

3. Starten Sie Lightroom neu.
