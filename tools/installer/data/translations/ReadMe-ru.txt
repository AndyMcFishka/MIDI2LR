=============
Предпосылки
=============

- Lightroom классика 11+
- Windows 10 или MacOS 12.0+
- USB MIDI-контроллер (например: Korg nanoKONTROL и т. Д.)

=====
Настроить
=====

1. Загрузите установщик для вашей операционной системы с https://github.com/rsjaffe/MIDI2LR/releases, где вы всегда найдете последнюю версию. При выборе папки для установки убедитесь, что вам не нужны права администратора для записи в эту папку - плагин может зависнуть, если он работает в ограниченной папке.

2. В Lightroom добавьте плагин MIDI2LR, для этого перейдите в «Файл» → «Менеджер плагинов» → «Добавить», перейдите и выберите папку MIDI2LR.lrplugin.

3. Приложение MIDI2LR должно запуститься автоматически. Если он закрыт, его можно запустить, выбрав «Файл» → «Дополнительные параметры модуля» → «Запустить MIDI2LR».

4. Перемещение любых контроллеров заполнит таблицу отображений и позволит вам установить функции. Вы также можете сохранять / загружать сопоставления.

5. Информация обо всех настройках и опциях находится в вики: https://github.com/rsjaffe/MIDI2LR/wiki.

================================
Обновление с предыдущей версии
================================

1. Выход из Lightroom.

2. Установите в тот же каталог, который использовался в предыдущей версии.

3. Перезапустите Lightroom.
