=============
förutsättningar
=============

- Lightroom klassiker 11+
- Windows 10 eller MacOS 12.0+
- En USB MIDI-controller (t.ex.: Korg nanoKONTROL, etc.)

=====
Uppstart
=====

1. Ladda ner installationsprogrammet för ditt operativsystem från https://github.com/rsjaffe/MIDI2LR/releaser, där du alltid hittar den senaste versionen. När du väljer en mapp för installation, se till att du inte behöver administratörsbehörighet för att skriva till den mappen - plugin kan krascha om den körs i en begränsad mapp.

2. I Lightroom lägger du till MIDI2LR-plugin genom att gå till Arkiv → Plugin-Manager → Lägg till och navigera till och välja mappen MIDI2LR.lrplugin.

3. MIDI2LR-applikationen bör automatiskt startas. Om den är stängd kan den startas genom att gå till File → Plug-in Extras → Start MIDI2LR

4. Att flytta alla kontroller fyller i kartläggningstabellen och låter dig ställa in funktioner. Du kan också spara / ladda mappningar.

5. Information om alla inställningar och alternativ finns i wiki: https://github.com/rsjaffe/MIDI2LR/wiki.

================================
Uppdatering från en tidigare version
================================

1. Avsluta Lightroom.

2. Installera i samma katalog som användes i den tidigare versionen.

3. Starta om Lightroom.
