=======
Warunki wstępne
=======

- Lightroom Classic 11+
- Windows 10 lub MacOS 12.0+
- Kontroler USB MIDI (np. Korg nanoKONTROL itp.)

=====
Organizować coś
=====

1. Pobierz instalator dla swojego systemu operacyjnego ze strony https://github.com/rsjaffe/MIDI2LR/releases, gdzie zawsze znajdziesz najnowszą wersję. Wybierając folder do instalacji, upewnij się, że nie potrzebujesz uprawnień administratora, aby zapisywać w tym folderze — wtyczka może ulec awarii, jeśli działa w folderze z ograniczeniami.

2. W Lightroomie dodaj wtyczkę MIDI2LR, przechodząc do Plik → Menedżer wtyczek → Dodaj i przechodząc do folderu MIDI2LR.lrplugin i wybierając go.

3. Aplikacja MIDI2LR powinna uruchomić się automatycznie. Jeśli jest zamknięty, można go uruchomić przechodząc do Plik→Plug-in Extras→Start MIDI2LR

4. Przesunięcie dowolnych kontrolerów zapełni tabelę mapowania i umożliwi ustawienie funkcji. Możesz także zapisywać/wczytywać mapowania.

5. Informacje o wszystkich ustawieniach i opcjach znajdują się na wiki: https://github.com/rsjaffe/MIDI2LR/wiki.
