=============
Conditions préalables
=============

- Lightroom classique 11+
- Windows 10 ou MacOS 12.0+
- Un contrôleur USB MIDI (ex: Korg nanoKONTROL, etc.)

=====
Installer
=====

1. Téléchargez le programme d'installation de votre système d'exploitation sur https://github.com/rsjaffe/MIDI2LR/releases, où vous trouverez toujours la dernière version. Lorsque vous sélectionnez un dossier pour l'installation, assurez-vous que vous n'avez pas besoin de privilèges d'administrateur pour écrire dans ce dossier - le plug-in peut se bloquer s'il s'exécute dans un dossier restreint.

2. Dans Lightroom, ajoutez le plug-in MIDI2LR en allant dans Fichier → Plugin-Manager → Ajouter et en naviguant et en sélectionnant le dossier MIDI2LR.lrplugin.

3. L'application MIDI2LR devrait se lancer automatiquement. S'il est fermé, il peut être lancé en allant dans Fichier → Extras de plug-in → Démarrer MIDI2LR

4. Le déplacement de n'importe quel contrôleur remplira la table de mappage et vous permettra de définir des fonctions. Vous pouvez également enregistrer / charger des mappages.

5. Les informations sur tous les paramètres et options se trouvent dans le wiki: https://github.com/rsjaffe/MIDI2LR/wiki.

================================
Mise à jour à partir d'une version précédente
================================

1. Quitter Lightroom.

2. Installez dans le même répertoire que celui utilisé dans la version précédente.

3. Redémarrez Lightroom.
