To use, download translations for each language with plugin tag from POEditor project as "strings" format, using default names. https://poeditor.com/projects/view?id=359585 Then run convert.sh to produce properly formatted and labelled txt files.

Then, remove those strings files, download translations for each language with app tag from POEditor project as "strings" format, using default names. https://poeditor.com/projects/view?id=359585 Then run apptrans.sh to produce properly formatted and labelled txt file.

Then, remove those strings files, download translations for each language with installer tag from POEditor project as "strings" format, using default names. https://poeditor.com/projects/view?id=359585 Then run installer.sh to produce properly formatted and labelled lng files.
