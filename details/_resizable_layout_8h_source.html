MIDI2LR  2.4.0.0
MIDI2LR is an application that interfaces MIDI controllers with Lightroom 6/CC. It processes MIDI input into develop parameter updates and photo actions, and sends MIDI output when parameters are changed for motorized feedback (on controllers that have motorized faders). A listing of available LightRoom commands is in the Wiki. Assistance on the code and design is welcome.

31 /**
32 Smoothly repositions and resizes child Components without rounding errors,
33 according to caller provided stretching parameters.
34 Usage:
35 1) Derive your Component from ResizableLayout
36 2) Give your Component a well defined default size
37 3) Add child Components and position them according to how
38 they should appear at the default.
39 4) For each child Component, also call addToLayout() and specify what
40 portion of the owner's growth or reduction should apply to each corner.
41 5) At the end of your constructor, call activateLayout() to turn it on
42 6) If you later manually reposition a control (for example, using a
43 ComponentDragger) call updateLayoutFor() on the control that moved,
44 or if you moved all the controls call updateLayout().
45 7) You do not need to override resized() to make this work, all you
46 have to do is lay things out when you create your child Components
47 and add anchors as appropriate.
48 Example
49 struct ContentComponent : Component, ResizableLayout
50 {
51 ContentComponent() : ResizableLayout (this)
52 {
53 // for layouts to work you must start at some size
54 // place controls in a location that is initially correct.
55 setSize(512, 384);
56 TextButton* b;
57 b = new TextButton ("Juce");
58 b->setBounds (32, 32, 512-64, 32);
59 // stretch to fit the width of the window
60 addToLayout (b, anchorTopLeft, anchorTopRight);
61 addAndMakeVisible (b);
62 b = new TextButton ("One");
63 b->setBounds (32, 96, 128, 32);
64 // take up 1/3 of the width of the window
65 addToLayout (b, Point<int>(0,0), Point<int>(33,0));
66 addAndMakeVisible (b);
67 // turn it on
68 activateLayout();
69 }
70 }
