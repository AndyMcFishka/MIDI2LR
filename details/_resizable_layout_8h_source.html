MIDI2LR  2.0.4.0
MIDI2LR is an application that interfaces MIDI controllers with Lightroom 6/CC. It processes MIDI input into develop parameter updates and photo actions, and sends MIDI output when parameters are changed for motorized feedback (on controllers that have motorized faders). A listing of available LightRoom commands is in the Wiki. Assistance on the code and design is welcome.

34 /**
35 Smoothly repositions and resizes child Components without rounding errors,
36 according to caller provided stretching parameters.
37 Usage:
38 1) Derive your Component from ResizableLayout
39 2) Give your Component a well defined default size
40 3) Add child Components and position them according to how
41 they should appear at the default.
42 4) For each child Component, also call addToLayout() and specify what
43 portion of the owner's growth or reduction should apply to each corner.
44 5) At the end of your constructor, call activateLayout() to turn it on
45 6) If you later manually reposition a control (for example, using a
46 ComponentDragger) call updateLayoutFor() on the control that moved,
47 or if you moved all the controls call updateLayout().
48 7) You do not need to override resized() to make this work, all you
49 have to do is lay things out when you create your child Components
50 and add anchors as appropriate.
51 Example
52 struct ContentComponent : Component, ResizableLayout
53 {
54 ContentComponent() : ResizableLayout (this)
55 {
56 // for layouts to work you must start at some size
57 // place controls in a location that is initially correct.
58 setSize(512, 384);
59 TextButton* b;
60 b = new TextButton ("Juce");
61 b->setBounds (32, 32, 512-64, 32);
62 // stretch to fit the width of the window
63 addToLayout (b, anchorTopLeft, anchorTopRight);
64 addAndMakeVisible (b);
65 b = new TextButton ("One");
66 b->setBounds (32, 96, 128, 32);
67 // take up 1/3 of the width of the window
68 addToLayout (b, Point<int>(0,0), Point<int>(33,0));
69 addAndMakeVisible (b);
70 // turn it on
71 activateLayout();
72 }
73 }
74 */
