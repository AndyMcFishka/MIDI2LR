MIDI2LR  3.0.0.7
MIDI2LR is an application that interfaces MIDI controllers with Lightroom 6+/CC Classic. It processes MIDI input into develop parameter updates and photo actions, and sends MIDI output when parameters are changed for motorized feedback (on controllers that have motorized faders). A listing of available LightRoom commands is in the Wiki. Assistance on the code and design is welcome.

32 /**
33 Smoothly repositions and resizes child Components without rounding errors,
34 according to caller provided stretching parameters.
35 Usage:
36 1) Derive your Component from ResizableLayout
37 2) Give your Component a well defined default size
38 3) Add child Components and position them according to how
39 they should appear at the default.
40 4) For each child Component, also call addToLayout() and specify what
41 portion of the owner's growth or reduction should apply to each corner.
42 5) At the end of your constructor, call activateLayout() to turn it on
43 6) If you later manually reposition a control (for example, using a
44 ComponentDragger) call updateLayoutFor() on the control that moved,
45 or if you moved all the controls call updateLayout().
46 7) You do not need to override resized() to make this work, all you
47 have to do is lay things out when you create your child Components
48 and add anchors as appropriate.
49 Example
50 struct ContentComponent : Component, ResizableLayout
51 {
52 ContentComponent() : ResizableLayout (this)
53 {
54 // for layouts to work you must start at some size
55 // place controls in a location that is initially correct.
56 setSize(512, 384);
57 TextButton* b;
58 b = new TextButton ("Juce");
59 b->setBounds (32, 32, 512-64, 32);
60 // stretch to fit the width of the window
61 addToLayout (b, anchorTopLeft, anchorTopRight);
62 addAndMakeVisible (b);
63 b = new TextButton ("One");
64 b->setBounds (32, 96, 128, 32);
65 // take up 1/3 of the width of the window
66 addToLayout (b, Point<int>(0,0), Point<int>(33,0));
67 addAndMakeVisible (b);
68 // turn it on
69 activateLayout();
70 }
71 }
72 */
