MIDI2LR

MIDI2LR is an application that translates MIDI Control Change commands into Lightroom actions. The wiki is read-only to prevent spamming. Contact me if you wish to add any pages or text. The wiki lists commands currently available through MIDI2LR. Instructions for use are in the wiki.

Latest Release

You can always download the latest release from https://github.com/rsjaffe/MIDI2LR/releases/latest. The wiki has a detailed listing of currently available commands.

Supported platforms

Mac OS X 10.9+ and Windows 7/8/10. Lightroom 6/CC.
